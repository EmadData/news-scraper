# News Scraper with Python

This project is a simple web scraping script written in Python.

## Features
- Scrapes news titles from Iranian news websites
- Categorizes news by topic
- Removes duplicate titles
- Extracts publish time
- Saves data to CSV file

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas
